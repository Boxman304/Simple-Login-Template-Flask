# Simple-Login-Template
This is a sample login template tested using postman where you create and authenticate users.
Database Used: MySQL
To use this login template create a new folder in your user files and create a flask Virtual Environment and proceed to install what is required (found in requirements.txt)
Make sure to access the directory in the project folder through the command prompt
The following must be written to create and activate the venv:
py -3 -m venv venv
venv\Scripts\activate
Afterthat, install flask and the requirements
flask run to run the server
Test using postman
